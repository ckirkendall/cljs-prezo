Example 1: JQuery Plugin (ClojureScript)

The Logger Function 
(ns cincijs.jquery.logger)

(defn logger [sel { pre :pre :or { pre "Log"}}]
  (.each sel #(let [obj (js/jQuery js/this)]
                (.log console pre obj))))

Calling Our Logger		 
(-> js/JQuery
    (logger { :pre "logger"})
    (.append "")
    (.text "hello"))
