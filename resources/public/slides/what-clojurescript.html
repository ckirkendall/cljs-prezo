What ClojureScript is and is Not?

ClojureScript is

  • Clojure!
  • A Lisp
  • Functional
  • Object Oriented
  • Simpler than JS
  • Safer than JS
  • More Extendable than JS

ClojureScript is Not

  • JavaScript - Unlike languages similar to CoffeeScript, ClojureScript is not a wrapper over JavaScript. It is a different language, with different constructs. Similar to how Clojure targets the JVM, ClojureScript is targeting JavaScript and the Google Closure library.
