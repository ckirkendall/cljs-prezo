Managing Upstream Dependencies

Should users have to worry about, a libraries, upstream dependencies?

Experimental Feature - deps.cljs

  • Place it in the root of your jar or zip.
  • This file allows you to specify :libs, :foreign-libs, and :externs that should be loaded from the classpath.


deps.cljs 
{
:foreign-libs [{:file "libs/my-foreign-lib.js"
                 :provides "org.compay.mylib"}]
 :libs ["goog/dom/query.js"]
 :extern ["js/jquery/jquery.js"]
 }
