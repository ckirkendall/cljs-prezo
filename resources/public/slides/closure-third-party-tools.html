Closure Third Party Tools

  • goog.dom.query (css3 selector used by Enfocus)
  • goog.dom.html (html parser)
  • goog.silverlight (silverlight.js compatible with compiler)
  • more info

These tools where developed by third party groups but are bundled with the Google Closure Library and use the "goog" namespace.
